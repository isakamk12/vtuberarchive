INTRODUCTION

アウトドア大好きな女子。ロードバイク、コスプレなど興味範囲が広く、トークと企画力で配信を回すタイプのライバー。関西弁の明るさと面倒見の良さから「オカン」扱いされやすいが、本人は20代であることを強く主張するのがお約束。

雑談・ゲーム・歌配信を軸に、突発ゲリラ配信も多い。前職で美容関係の仕事経験があり、コスメや美容知識にも強い。陽キャな表向きと、BLを含む濃いオタク気質のギャップが魅力として定着している。

CHARACTER TRAITS

MC・司会進行力

コミュ力が高く、複数人の場でも会話を回して落とし所を作れる。企画配信の成立力が高い。

アウトドアと多趣味

ロードバイク、旅行、写真、文房具など趣味の幅が広い。日常系トークの引き出しが多い。

美容知識と実用性

ビューティーアドバイザー経験を活かし、美容や化粧品の話題に強い。実用系の相談にも応えやすい。

オタク・腐女子面

BLへの情熱や作品語りの深さを持ち、陽キャイメージとの差で強い個性が出る。

RECORDS

早瀬とサシ飲み

ゲストと1対1で本音トークを引き出す恒例企画。2025年11月には1ヶ月連続企画として拡張された。

チューリップ組

健屋花那・シェリンと同期ユニットを形成。3人の距離感と役割分担がコンテンツになっている。

龍が如く愛

シリーズへの熱量が非常に高く、実況でも推し語りでも強いエネルギーを見せる。

嘘プロフィール挨拶

新人の初投稿に合わせて“嘘プロフィール”で絡みに行く恒例行事があり、箱内コミュニケーションの名物になっている。
