DESCRIPTION

「だつりょく系シンガー」、みにくいアヒルの子（白鳥の雛）。絵本の世界から人間界へやってきた。心の傷を隠すために包帯を巻いており、命の恩人である神「とり」に見守られながら生活している。ウィスパーボイスから力強いアニソンまで歌いこなす。

EPISODES

「えさ」たちとの不思議な絆

リスナーの総称は「えさ」。待機所では「ON SETと書かれた椅子」のステッカーを投げる「待機イス」という独自文化がある。リスナーを「エレベーター」に例え、いつでも同じ目線で寄り添うエモい一面も。

狂気的な「ペルソナ」＆「アイマス」愛

ゲーム『ペルソナ』では感情移入しすぎて号泣。配信外で進めすぎたあまり、「未来から来た」と称して録画を流し自身はチャットに降臨。アイマス曲では自身の「プロデューサー人格」が暴走することも。

多才なクリエイター気質

イラスト制作はプロ級。春野氏からの楽曲提供を受けた際は限界オタクっぷりを発揮。自分の声や絵に自信がないと言いつつも、表現活動への情熱は本物。

異例の移籍と10万人達成

個人勢からRe:AcTへ名前とガワを保ったまま移籍するという異例の決断。着実にファンを増やし、10万人記念ライブでは感謝の歌を響かせた。
