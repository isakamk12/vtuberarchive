序論：Re:AcTの組織的背景とバーチャルタレント業界における位置付け

株式会社mikaiが運営するバーチャルタレント事務所「Re:AcT（リアクト）」は、2018年の設立当初「KAGAYAKI STARS」として始動し、その後、音楽活動を主軸に据えた現在の形態へと進化を遂げた歴史を持つ 1。本報告書は、Re:AcTに所属する全タレントの現況、契約ステータス、同期関係、および音楽レーベル編成について、アーカイブ構築に耐えうる精度で詳細に分析・記述するものである。

バーチャルタレント（VTuber）業界が成熟期を迎える中で、Re:AcTは「バーチャルで、自分らしく生きよう」というスローガンを掲げ、単なるライブ配信者ではなく、アーティストとしての側面を強化する戦略を採ってきた 2。この戦略の根幹をなすのが「レーベル制」の導入であり、タレントの歌声やキャラクターの特性に合わせて「ReAliz」「kaleidscope」「sinkirow」「Parfait」といった複数のレーベルに再編成を行うことで、ターゲット層の明確化と音楽的クオリティの向上を図っている 3。2025年からは、従来の音楽路線に加え、eスポーツ特化部門「Re:AcT Gaming」の拡大や、多人数にわたる新規タレントの相次ぐデビューなど、組織の規模を劇的に拡大させている 4。

現役所属タレントの網羅的調査および活動分析

現在のRe:AcTは、メインブランチ、Re:AcT Gaming、および新設されたレーベルに分かれる重層的な構造を有している。以下に、2025年から2026年現在における全現役メンバーの基本データと活動ステータスを詳述する。

メインブランチおよび主要所属メンバー

メインブランチに属するタレントは、Re:AcTのブランドイメージを象徴する存在であり、その多くが強力な歌唱力や独自の配信スタイルを確立している。

これらのメンバーは、Re:AcTの黎明期から中期にかけて加入し、現在の事務所の基盤を築いてきた。特に獅子神レオナや花鋏キョウは、バーチャルシンガーとしての知名度が極めて高く、事務所の顔としてフェスやライブイベントの主軸を担っている 2。

2025年デビュー組：次世代タレントの台頭

2025年はRe:AcTにとって「第二の創業期」とも呼べる拡大の年となった。3月の第1弾デビューを皮切りに、6月と7月にはさらに5名のメンバーが加わり、一挙に8名の新戦力が投入された 4。

この新世代の導入により、Re:AcTは既存の「歌」に加えて「クリエイティブ（アニメ制作など）」や「配信ボリューム（長時間配信）」といった新たな強みを獲得しようとしていることが伺える 2。特に音羽奏來や不知名イヲのように、特定の楽器演奏や物語性を重視したメンバーの加入は、初期の音楽路線をより高度化させる意図があると考えられる。

Re:AcT Gaming（RAG）の組織的役割と構成

eスポーツへの注力は、現在のVTuber事務所にとって避けて通れない戦略的選択肢である。Re:AcTも「Re:AcT Gaming（RAG）」という特化部門を設立し、音楽活動とは異なるファン層の獲得に成功している 2。

RAGのメンバーは、単にゲームをプレイするだけでなく、大会への出場や技術向上を目的とした活動を主としており、事務所内でも「アスリート」としての側面が強調されている 5。これにより、Re:AcTは「音楽のRe:AcT」と「ゲームのRAG」という二面性を持つ強固なエンターテインメント組織へと変貌を遂げた。

契約状況と卒業・移籍タレントの歴史的経緯

アーカイブにおいて最も重要なのは、活動を終了したタレントの履歴と、その背後にある契約上の理由である。Re:AcTは、初期の男女混成時代から、現在の女性専門事務所への移行過程で、戦略的な再編を繰り返してきた 1。

卒業・引退および契約終了タレント一覧

Re:AcTの卒業生は、公式に「卒業」として発表されるケースが大半であり、不祥事による契約解除というよりも、活動の節目やユニットの解散に伴うケースが多い。

男性部門（SuiSay/METEO）とSTeLLaへの移籍

Re:AcTはかつて男性バーチャルアイドルユニットを擁していたが、2018年から2021年にかけて大きな転換点を迎えた。

SuiSayの解散: 2018年12月31日をもって、折緒コウヤ、美影ルクス、星屑れんの3名が卒業し、ユニットは解散した 1。これは事務所が初期の多角化戦略を見直し、運営資源の集中を図り始めた最初の事例である。

STeLLaへの移籍: 残留した一部の男性タレント（湊音みなみ、皇噛ユカリ等）は、社内の別ブランド「STeLLa」へと移籍した 1。これによりRe:AcT本体は女性VTuber専門の事務所へと純化した。

METEOの解散: STeLLa移籍後も活動を続けていたMETEO（唯牙コハク、皇噛ユカリ）も、2021年4月に全メンバーが卒業し、プロジェクトは終了した 1。

これらの動向は、Re:AcTが「アイドルの総合商社」から「女性アーティスト・特化型レーベル」へと明確なポジショニングの変更を行ったことを示唆している。

音楽レーベル制の深掘り：戦略的意図と音楽的背景

Re:AcTのアイデンティティは、2023年に再編された4つの音楽レーベルに集約されている。これはタレントを単なる配信者ではなく、各ジャンルの専門アーティストとして扱うための仕組みである 3。

各レーベルのコンセプトとリリースの傾向

1. ReAliz（リアライズ）

コンセプト: パワフル、情熱、コール＆レスポンス、ライブの熱狂 3。

主要メンバー: 獅子神レオナ、水瓶ミア、皇ロゼ。

分析: このレーベルは事務所の「動」を司る。獅子神レオナの『History:E』のようなアルバムや、水瓶ミア、皇ロゼとのユニット曲『Proto Rebirth』など、ロックやアップテンポな楽曲が主流である 2。

2. kaleidscope（カレイドスコープ）

コンセプト: 繊細、透明感、儚さ、芸術性 3。

主要メンバー: 花鋏キョウ、稀羽すう、（旧所属：かしこまり、宇佐美ユノ）。

分析: Re:AcTの「美」を象徴するレーベルである。花鋏キョウの『hasamiUta』シリーズや、稀羽すうの『Mornin' Bell』など、歌唱力を最大限に活かしたバラードや情緒的な楽曲が特徴である 2。

3. sinkirow（シンキロウ）

コンセプト: ゴシック、幻想的、魅惑、酸いも甘いも知る大人の魅力 3。

主要メンバー: 丸餅つきみ、猫乃ユキノ、九楽ライ、（旧所属：姫熊りぼん、射貫まとい）。

分析: ストーリー性や独特の世界観を重視する。九楽ライの『白と枷』や、姫熊りぼんの『秘密ノ花園』のように、ダークでミステリアスな、あるいは物語の1ページを切り取ったような表現が多用される 6。

4. Parfait（パルフェ）

コンセプト: “カワイイ”の追求、元気、あどけなさ、リスナーへの癒やし 3。

主要メンバー: 天川はの、夢川かなう、月紫アリア。

分析: 最も新しいレーベルであり、現代の「アイドルVTuber」像に最も近い。天川はのの『ラブプライズ』や、月紫アリアの『あたし争奪戦トーナメント』など、キャッチーで中毒性のあるポップソングを主軸としている 2。

ユニットおよび全体曲の変遷

事務所全体の連帯感を強めるため、レーベルの枠を超えたユニットや、所属メンバー全員が参加する全体曲も重要な位置を占めている 6。

このような楽曲リリースは、ファンに対して「個」の魅力だけでなく「組織」としての魅力を提示し、いわゆる「箱推し」の文化を育成する causal（因果的）な役割を果たしている。

同期関係とユニットの系譜：アーカイブ構築の鍵

VTuberのアーカイブにおいて、同期という括りはファンのコミュニティ形成に多大な影響を及ぼす。Re:AcTにおける主要な同期・ユニット関係を整理する。

黎明期：第1期・第2期（旧KAGAYAKI STARS）

獅子神レオナ、射貫まとい（卒業）、花鋏キョウらが相次いでデビューしたこの時期は、Re:AcTの骨格が作られた時期である。特に「すたーべあ！」（姫熊りぼん、天川はの）のような、個別のユニット活動が先行する傾向があった 1。

中期：多様化の時代

2020年から2021年にかけて加入した夢川かなう、丸餅つきみ、月紫アリア、猫乃ユキノらは、配信スタイルの多様化をもたらした。彼女たちは後にレーベル再編の際に、それぞれの個性に合った場所（Parfaitやsinkirow）へと振り分けられた。

転換期：九楽ライ・皇ロゼ・稀羽すうの加入

2021年後半から2022年にかけて加入したこれらのメンバーは、現在のRe:AcTの音楽性の洗練に大きく寄与した。特に稀羽すうの加入は、kaleidscopeの層を厚くし、Vsingerとしてのブランディングを確固たるものにした 2。

拡大期：2025年大規模採用

白音ゆきら「2025年3月組」と、音羽奏來ら「2025年6/7月組」は、Re:AcT史上最大規模の増員であり、これにより事務所は中規模から大規模なプロダクションへとその性質を変貌させた 4。

今後の展望と2026年へ向けた戦略的示唆

Re:AcTの現状のデータからは、今後の持続的な成長に向けた3つの明確なベクトルが観察される。

第一に、音楽の専門化である。レーベル制の導入と新人の音楽的スキル（シンガーソングライター等）の重視は、一時的な流行に左右されない「アーティストブランド」の確立を目指している 3。 第二に、メディアミックスの多角化である。公式VISAカードの発行、大規模フェス「Re:AcTフェス2025」の開催、クリスマスライブの全メンバー出演など、配信以外の接点を増やすことで、ファンのロイヤリティを高めている 2。 第三に、eスポーツ領域の定着である。RAGの継続的な新メンバー投入は、ゲーム実況がVTuber文化の中心であり続ける限り、Re:AcTが市場での存在感を維持するための防波堤となっている 5。

契約状況についても、過去の男性部門の整理や、ユニットの解散を伴う卒業といった「痛みを伴う改革」を経て、現在の女性タレントを中心とした安定した運営体制に移行したことは、組織としての成熟を示している。ククルア・クレイユの卒業発表などは、長年活動する組織であれば避けられない事象であるが、それを公式な「お知らせ」として適切に処理し、新陳代謝を繰り返しながら組織を維持する仕組みが機能している 2。

結論

本報告書におけるRe:AcTの所属タレントおよび組織構造の調査は、同事務所がバーチャルタレント業界において、音楽を核とした極めて独自性の高い進化を遂げたことを明らかにしている。初期の「KAGAYAKI STARS」から現在の「Re:AcT」への改称、男性タレントの分離、そしてレーベル制への移行という一連の流れは、すべてが「音楽を通じたアイデンティティの確立」という一貫した目的のために行われてきたものである。

現役の全タレントは、それぞれが持つ音楽的・配信的特性に基づいて最適なレーベルに配置されており、2025年の大規模な増員は、その既存の枠組みをより強固にし、表現の幅を広げるための戦略的補強であると結論付けられる。アーカイブ構築においては、本報告書で示したタレントの加入・卒業の履歴、同期関係、そしてレーベルごとのリリースの推移を基軸とすることで、Re:AcTという組織の全体像を正確に記述することが可能となる。

Re:AcTは、単なるタレント事務所の枠を越え、バーチャルな存在が「自分らしく」アーティストとして生きるためのプラットフォームへと進化を続けており、2026年以降のさらなる飛躍に向けた基盤は十分に整っていると言える。
