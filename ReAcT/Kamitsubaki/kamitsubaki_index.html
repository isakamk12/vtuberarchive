OVERVIEW

KAMITSUBAKI STUDIOにおけるバーチャルアーティスト・エコシステムの包括的アーカイブ。日本のエンターテインメント業界、特にインターネットカルチャーを起点とする領域において、株式会社THINKRが運営する「神椿スタジオ」は、既存の「VTuber事務所」という定義を根底から覆す、独自のクリエイティブ・エコシステムを構築してきた。

STRUCTURE

2019年の発足以来、同スタジオは「バーチャルシンガー」や「バーチャルアーティスト」という呼称を意図的に選択し、キャラクターの背後に存在する実在の歌唱者の個性を尊重しつつ、物語とテクノロジーを融合させた多層的な表現を追求している。2023年にSINSEKAI STUDIOを統合し、現在は4つの専門DIVISIONによって構成される巨大なクリエイティブ・ギルドへと進化を遂げた。
