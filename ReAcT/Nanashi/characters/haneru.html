DESCRIPTION

ななしいんく（旧774 inc.）を象徴するリーダー的存在。有閑喫茶あにまーれのリーダーとしてデビューし、現在はプロダクション全体を牽引する。卓越した企画力とトークスキルを持ち、サンリオとのコラボや大型対局番組の主催など、多方面で活躍している。
