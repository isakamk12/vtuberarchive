AGENT // Lupinus Virtual Games (LVG)

GAME STYLE & ACHIEVEMENTS

FPSゲームはもちろんのこと、麻雀（雀魂）が好きでよく配信している。

STREAMING STYLE

「なずNEWS」というニュース番組風の定期配信や、「なずクッキング」「星座占い」などのコーナー企画を行い、メンバーの裏話やリスナーからのタレコミを面白おかしく紹介していた実績がある。

生活リズムが不規則で深夜や早朝にゲリラ配信を行うこともあり、「いつ寝ているのかわからない」と言われる過激な配信スタイルも。

PERSONALITY & TRIVIA

花芽すみれの実の妹。明るく元気な性格で、グループ内の「おバカ（アホ）担当」と言われることもあるが、実は頭の回転が速く、企画力に長けている。

姉のすみれとは配信上でよく言い合いをするが、根底には強い信頼関係がある仲良し姉妹。
